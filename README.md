# Radzuweit Analyse Website
This repository contains the source for a static Next.js website.

## Development

Install dependencies and start a local development server:

```bash
npm install
npm run dev
```

## Deployment

Build the project and export the static site:

```bash
npm run build
```

The exported files will appear in the `out/` directory. Copy the contents of
this folder to the repository root (or configure GitHub Actions) to update the
GitHub Pages site.
